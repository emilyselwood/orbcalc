Orb Viewer

This is a model of the entire solar system with every known asteroid marked. Each point shows the real position of an asteroid. The lines show the orbits of the major planets.

Left click and drag to look around. Right click and drag to move around. Scroll to zoom in and out.

There is a 'T' button in the top right next to the help button that will enable a flight through the solar system.

The data is sourced from the Minor planets center and processed and visualised by Wil Selwood and collaborators using Orb Calc to give positions. This project is open source and can be found on github
